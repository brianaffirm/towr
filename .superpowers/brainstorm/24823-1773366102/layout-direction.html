B. Command Center

Big number counters at top. Compact list on left, large terminal panel on right. Optimized for monitoring — like Datadog or Grafana. Dense but clear.

C. Timeline Feed

Vertical timeline showing all workspaces in chronological order. Most recent at top. Like a GitHub activity feed. Great for "what happened overnight" but less useful for real-time monitoring.

Click a layout to select it. Consider: which one would you check on your phone at 7am to see if the overnight run succeeded?
